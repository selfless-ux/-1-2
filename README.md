# Слайдер изображений

Простой слайдер изображений, реализованный на HTML, CSS и JavaScript (vanilla JS).

## Функции

- Переход по кнопкам «Вперед» / «Назад»
- Автоматическая смена изображений каждые 4 секунды
- Плавный эффект fade при смене слайдов
- Пауза при наведении мыши на слайдер
- Счётчик текущего изображения (например, "Изображение 2 из 5")
- Возможность кликнуть на картинку для перехода вперёд
- Адаптивный дизайн для мобильных устройств и планшетов
